Theory

These samples consist of three GitHub repositories. In addition, they are piped through jsFiddle, an online code editor from Mozilla. Presenting code samples this way makes them easy to share, combine and modify.

Rhizome Project

Rhizome is a set of libraries I am developing related to mobile health. These are templates and code samples that demonstrate how social aspects like Questionnaires and Careplans can be enhanced to support better patient applications for health.

Folds Samples

Literary samples using AngularJS, a JavaScript from Google which allows custom definition of HTML tags. Angular is an example of "Web Componentization", an emerging approach in web applications, which blurs the line between author anddeveloper.

Open Data

Providing better access to information requires exposing data in a way that can be easily used, and then providing tools with which it can be manipulated. In these examples, I have used the Data Driven Documents (d3.js) library to show how this might look.

Each of these repositories is an ongoing project, involving creating and accumulating sample code.
